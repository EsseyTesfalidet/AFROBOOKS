# AFROBOOKS

**Africa's boldest ebook marketplace** — discover, buy, and read African ebooks online.

🌍 **Live:** [afrobs.com](https://afrobs.com)

---

## About

AFROBOOKS is a production ebook marketplace built for African authors and readers. Authors publish and sell their work. Readers discover, purchase, and read directly in the browser — no downloads, no third-party apps.

> This is a proprietary application. The source code is shared for portfolio purposes only — it is not open source and may not be cloned, forked, or reused.

---

## Features

**For readers**
- Browse and search ebooks by genre
- Free sample chapters before purchasing
- In-browser reader with reading progress sync
- Pre-order upcoming releases
- Subscription plan for unlimited reading
- Wishlist, library, and purchase history
- Referral program

**For authors**
- Multi-step publish flow with chapter editor
- Free preview / locked chapter control
- Sales analytics and earnings dashboard
- Stripe Connect payouts
- Co-author revenue splitting

**Platform**
- Admin dashboard — moderation, payouts, user management
- Report system for flagging content
- Verified purchase badges on reviews
- Installable as a PWA on mobile and desktop
